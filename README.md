# 📊 Análise de Dados de E-commerce

## 📌 Objetivo
Realizar uma análise exploratória de dados de um e-commerce, com foco em identificar padrões de vendas, comportamento de consumo e relações entre variáveis.

---

## 📂 Dataset
Os dados utilizados contêm informações sobre:
- Quantidade de produtos vendidos
- Preço
- Marca
- Avaliações e descontos

---

## 🛠️ Ferramentas utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 📊 Análises realizadas

### 🔹 Distribuição de vendas
- A maior parte das vendas está concentrada em valores mais baixos
- Presença de outliers com volumes muito elevados

### 🔹 Relação entre preço e vendas
- Produtos mais baratos tendem a ter maior volume de vendas
- Relação negativa leve entre preço e quantidade

### 🔹 Vendas por marca
- Forte concentração de vendas em poucas marcas
- Destaque para Zorba e Lupo

### 🔹 Correlação entre variáveis
- Forte relação entre número de avaliações e quantidade vendida
- Baixa influência de preço e desconto

---

## 📈 Conclusão
A análise demonstra que o volume de vendas está mais relacionado à popularidade dos produtos do que ao preço ou desconto. Além disso, observa-se uma concentração de mercado em poucas marcas, indicando possível dominância.
